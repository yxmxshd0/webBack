# Backend для курса Web-разработки в Московском Политехе

## Запуск ##
Склонируйте проект
```https://github.com/yxmxshd0/webBack.git```  
### Обычный способ ###
Установите зависимости  
```npm install```    
Запуск  
```npm run start```   
или  
```npm run test``` - для разработки
